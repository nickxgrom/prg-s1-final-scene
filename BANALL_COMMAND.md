# Команда для бана всех игроков

## Использование

Для бана всех игроков (онлайн и оффлайн) на сервере используйте команду:

```
/banall
```

## Как выполнить

### Из консоли сервера:
```
banall
```

### Из игры (требуется OP или разрешение):
```
/banall
```

## Что делает команда

1. **Банит всех онлайн игроков**:
   - Кикает их с сервера с красивым сообщением
   - Добавляет в бан-лист с причиной "Сезон завершен. Спасибо за игру!"

2. **Банит всех оффлайн игроков**:
   - Проходит по всем игрокам, которые когда-либо заходили на сервер
   - Добавляет их в бан-лист (если они еще не забанены)

3. **Выводит статистику**:
   - Количество забаненных онлайн игроков
   - Количество забаненных оффлайн игроков
   - Общее количество забаненных игроков

## Сообщение при бане

Игроки увидят следующее сообщение:

```
Сезон завершен

Спасибо за игру!

Увидимся в следующем сезоне!

https://prgcraft.evenfine.name/
```

## Разрешения

- Команда доступна операторам сервера (OP)
- Или игрокам с разрешением: `prgcraft.banall`

## Примечание

⚠️ **ВНИМАНИЕ**: Эта команда банит ВСЕХ игроков на сервере без возможности отмены через команду. Для разбана потребуется использовать стандартные команды Minecraft (`/pardon <игрок>`) или редактировать файл `banned-players.json`.
